# fenixBasketApp
CMS za potrebe Fenix Basket kluba

<b>frontend</b>: HTML + CSS + JS (Axios za slanje HTTP requestova)<br>
<b>backend</b>: PHP + MySQL (MariaDB)
  
<br>
Potrebne dodatne biblioteke: <b>axios</b>, <b>parcel</b>
